# Sales Analytics Dashboard

## Overview
This project presents an interactive Sales Dashboard built to analyze e-commerce sales performance across multiple dimensions including product category, customer demographics, and time trends.

The dashboard enables stakeholders to quickly monitor key performance indicators and uncover actionable insights.

---

## Key Metrics

- Total Revenue: €649.02K
- Total Quantity Sold: 11,997
- Total Orders: 2,400
- Average Product Price: €53.83
- Average Customer Rating: 4

---

## Dashboard Features

- Sales trend analysis over time
- Top performing product categories
- Customer demographic analysis
- Interactive filtering by:
  - Gender
  - Order Mode
  - Week
  - County

---

## Key Insights

- T-Shirts and Jeans are the top-selling products.
- Female customers contribute to over 50% of purchases.
- Sales peak around week 10, suggesting seasonal demand patterns.
- Multiple order channels drive customer engagement.
